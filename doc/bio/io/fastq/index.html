fastq

Module bio::io::fastq [-]  [+] [src]

An iterable FASTQ parser. The io modules of Rust are currently rewritten. Hence, this modules will be subject to changes soon.

Example

use bio::io::fastq::FastqFile;
use std::old_io::{BufferedReader, stdin};

let mut buffer = BufferedReader::new(stdin());
let fastq = FastqFile::new(buffer);
